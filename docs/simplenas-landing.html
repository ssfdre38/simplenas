✨ Features

Everything You Need

Powerful NAS management tools designed for simplicity and performance

🗄️

ZFS Pool Management

Create, monitor, and manage ZFS RAIDZ pools with an intuitive web interface. Real-time status updates and health monitoring built-in.

📊

Real-time Monitoring

Track CPU, memory, disk usage, and network statistics with beautiful dashboards. Stay informed about your system's health at a glance.

🔐

Secure Access

Built-in authentication system with user management. Keep your data safe with role-based access control and secure sessions.

🚀

Lightweight & Fast

Minimal resource footprint means SimpleNAS runs smoothly even on modest hardware. Written in C# for maximum performance.

🌐

Web-based UI

Access your NAS from anywhere with a modern, responsive web interface. Works perfectly on desktop, tablet, and mobile devices.

🐧

Linux Native

Optimized for Ubuntu and Debian systems with full ZFS support. Native Linux integration for maximum compatibility.

⚡ Quick Start

Get Running in Minutes

Deploy SimpleNAS on your server with just a few commands

terminal 
# Clone the repository
git clone https://github.com/ssfdre38/simplenas.git
cd simplenas/SimpleNAS

# Build and publish
dotnet publish -c Release -o publish

# Run SimpleNAS
cd publish
./SimpleNAS

# Access at http://localhost:8000
# Default login: admin / SimpleNAS2026
