Quick Start

Get SimpleNAS up and running in minutes:

# Clone the repository
git clone https://github.com/ssfdre38/simplenas.git
cd simplenas/SimpleNAS

# Build and publish
dotnet publish -c Release -o publish

# Run SimpleNAS
cd publish
./SimpleNAS

Access the dashboard at http://localhost:8000
Default credentials: admin / SimpleNAS2026
